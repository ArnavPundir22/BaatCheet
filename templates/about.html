About BaatCheet

Ephemeral, Instant Video & Text Chat

BaatCheet is a next-generation ephemeral chat platform designed for spontaneous, secure, and private communication. Built on the principle that not everything needs to be saved forever.

🔒 The Zero-Log Guarantee

BaatCheet does not use a permanent database. All chat rooms, text messages, and connections are held entirely in volatile server memory. When the last participant leaves, the server executes a self-destruct sequence, instantly wiping the room and all its data from existence. No SQL. No logs. No history.

Core Features

  • 🎥 P2P Video & Audio: High-quality, low-latency WebRTC streams.
  • 👻 View-Once Media: Send audio notes and images that instantly self-destruct after being viewed or played once.
  • 📱 Mobile First PWA: Fully installable Progressive Web App with a custom Picture-in-Picture (PIP) miniplayer for multitasking.
  • ✨ Cyberpunk Aesthetics: A stunning, responsive glassmorphism UI with smooth CSS animations.

Under The Hood

Powered by Python & Flask on the backend, utilizing Socket.IO for real-time WebSocket signaling, and Redis for highly-scalable, ephemeral state management. The frontend is built entirely with pure, vanilla JavaScript and CSS3 for maximum performance without the bloat of frontend frameworks.
